Angular

angular是MVC框架，优点是利于模块化，可以复用。

使用npm下载Angular，命令：npm install angular

将angular粘贴到script文件夹中

第一节：绑定数据

引入<script src="script/angular.js"><script>

<body ng-app="">在这里规定范围是入口

<input type="text" ng-model="txt">

<h1 {{txt}}></h1>绑定了input与h1，在文本框中输入的内容同时在h1中显示

此时的引入放到下面会先加载页面，同时标记会闪烁一下，但若放到上面若页面又会影响速度。解决方案是使用标签内部绑定ng-bind="txt"如：h2标签里面写字套用span标签，在span标签内加入绑定。
