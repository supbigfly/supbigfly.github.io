js变量

变量分为全局变量和局部变量，都在script标签内，在函数内的是局部变量，外的是全局变量，局部变量只能在函数内起作用，而全局变量在哪里都起作用。想要得到局部变量的数值得定义全局变量通过调用函数将局部变量数值赋值给全局变量然后通过alert等显示。代码如下：
function demo(){
var num1 = 10;//此时的num1是局部变量
num3 = 4;无关键字var的变量是全局变量如需要num3的值即使将return注释掉也能拿到
return num1;
}
var num2 = demo();//此时的num2是全局变量
alert(num2);//不想使用return须将alert写到函数里面
