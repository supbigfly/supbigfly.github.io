Linux deepin atom

今天我鼓起勇气来到了金梦海湾，见到了peter，很高兴和大家一起相处，互相学习，互帮互助。

首先，安装linux系统，在宿舍捣腾了两天没搞定。在大家的帮助下安了deepin。后来，浩哥来了，发现他装的是ubuntu
这让我羡慕不已。我的电脑在关机的一刹那总会听到很微小的“啪儿”声响，于是借用浩哥的系统选择了重新安装，到了下午问题又出来
了,我的安装跟大家的操作不一样，无奈只好做个无用功装回去吧。
①分区表混乱错误，使用分区工具修复。
②ctrl+alt+T打开终端
③sudo apt-get install git curl(sudo最高权限)
④git-h(帮助信息)
⑤sudo apt-get install atom(安装atom程序)
⑥atom .(在当前文件夹下使用atom打开所有文件)
⑦git diff查看当前修改什么
⑧git log查看日志
⑨git config--list

在atom下安装emmet插件
