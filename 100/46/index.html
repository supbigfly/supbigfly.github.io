001-介绍实现输出javascript是互联网最流行的脚本语言，与java没什么关系，这门语言可用于web和HTML，更可广泛用于服务器、pc端、移动端。是一种轻量级的编程语言，可插入HTML页面并且所以浏览器都支持

html中的脚本必须位于<script></script>标签之间脚本可被放置在html页面的<body>和<head>部分中，一般放置在<head>中不限制脚本数量。语法如下：document.getElementById.innerHTML

002-语法注释

javascript语句向浏览器发出命令，告诉浏览器做什么，执行顺序按编写顺序，标识符以字母下划线开始对大小写敏感单号注释//多行注释/**/语法如下document.write(1)document.write(2)先写1再写2

003-变量和数据类型

变量是来储存信息的容器，声明变量通过关键字var变量名称值的顺序来进行。数据类型：①字符串String②数字Number③布尔Boolean④数组array⑤对象object⑥空null⑦未定义⑧通过赋值null清除变量。数组定义：①var arr = [1,2,3];document.write("arr[0],arr[1],arr[2]");②var arr = new array["我是狗","我是猪","我是驴"];document.write(同上)③var arr = new array();arr[0] = 10;arr[1] = "我";
